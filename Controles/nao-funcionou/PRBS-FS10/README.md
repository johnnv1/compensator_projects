# Projeto PRBS com frequencia 0 que não funcionou

Os controladores encontrados nas pastas subsequentes não funcionaram na planta real, porem em nivel de simulação eles funcionam quando utilizado eles acima de uma determinada frequencia (insto pq ao aumentar a afrequencia o controlador entra no circulo unitario). Porem nos testes das plantas reais os mesmo não funcionaram.

Ao analiza-lo foi encontrado erros de projetos ao determina-los, isto é as especificações, de erro, tempo de estabilização e etc, não condiziam e/ou não estavam coerentes com a planta identificada.
